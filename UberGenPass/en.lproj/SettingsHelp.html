Master Password

Type your SuperGenPass master password into the two Master Password text fields. The status indicator between the fields changes color as you type. Green indicates that the two passwords match. Red indicates that one or more characters do not match. Yellow indicates that the fields match so far, but one field is shorter than the other.

UberGenPass asks for your master password again when you relaunch it while it's running in the background. In this case, it knows the master password that you typed last time. If you retype the same text into the left password field, it automatically fills in the right field for you.

Tip: You may find it easier to enter the passwords with the device rotated to landscape.

Save Password Hash

When you first launch UberGenPass, it always asks for your master password. Since it never stores the master password on the device for security reasons, it can't automatically fill in the right field for you as described above. Typing the master password twice reduces the chance of a mistake, but it can be tiresome. If you enable Save Password Hash, UberGenPass saves a SHA-256 hash of your master password in the secure iOS Keychain. In this case, as soon as the hash of the text you type in the left password field matches the saved hash, it fills in the right field for you.

Save Recent Sites

If you enable Save Recent Sites, UberGenPass keeps a list of recent sites for which a password has been generated. These sites are used to populate a suggestion list that updates as you type a site name. Recent sites are stored in the secure iOS Keychain and cleared when you disable the feature.

Background Timeout

The Background Timeout setting determines how long UberGenPass can be running in the background before asking for your master password again when relaunched. By default, this is set to the most secure option, "immediately". You can optionally suppress this for up to five minutes.

Add Safari Bookmarklet

If you tap Add Safari Bookmarklet, UberGenPass will open a page in Safari describing how to integrate the two apps for faster workflow. Adding a bookmarklet to Safari lets you launch UberGenPass directly from a web page with the URL of the page already filled in for you, and also return to the page in Safari directly from UberGenPass.
